# simple-webauthn

## 説明

この Repository は Passkeys を学習するために作成されました。

## 実行方法

### 1. tool install

```bash
$ make install
```

### 2. build container

```bash
$ make build
```

### 3. start container

```bash
$ make up
```

### 4. lets play passkeys

-   front: http://localhost:3000
-   back: http://localhost:8080
